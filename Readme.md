Lab A: Pathfinding Problem

How to check the goal test
1/ In maze class use prizesCount function to update the number of prizes left in the maze
2/ After that, function goalTest will return True if there is no prizes left else return False

How to run search algorithms on each input maze file
1/ Open (or compile) file main.py
2/ Run file and follow the instructions show on the screen
3/ Select the txt file to execute
4/ Select the search algorithm for the txt file
	Note: For txt files that start with "multiprize...", you don't need to select search algorithm
5/ An output file will appear on the folder, open it to see the result.
